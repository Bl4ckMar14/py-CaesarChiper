# Caesar Cipher Python

Simple Caesar Cipher tool built from scratch using Python.

## Features

- Encrypt text
- Decrypt text
- Supports uppercase letters
- Supports lowercase letters
- Preserves spaces, numbers, and symbols

## How to run

```bash
pip install -r requirements.txt
```

```bash
python3 caesar_cipher.py
```

## Example

Mode:

e

Plaintext:

HELLO WORLD

Shift:

3

Output:

KHOOR ZRUOG
